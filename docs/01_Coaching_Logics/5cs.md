# 2. The 5 Cs (優れた選手の定義)

英国のサッカーアカデミー等で採用されている指標です。
**「バスケットボール選手を育てる前に、人を育てる」** という考え方に基づきます。

## 🌟 5つの要素（スターチャート）

| 要素 | 英語 (English) | 意味・旭町での定義 |
| :--- | :--- | :--- |
| **1** | **Competence** | **競技能力**<br>技術、戦術理解、フィジカル。<br>👉 *5Out戦術で「個の力」を磨く* |
| **2** | **Confidence** | **自信**<br>自己肯定感。ミスしても立ち直る力。<br>👉 *コーチは「ナイスチャレンジ」と声をかける* |
| **3** | **Connection** | **つながり**<br>仲間との絆、チームへの帰属意識。<br>👉 *「全員攻撃（GUN）」で一体感を作る* |
| **4** | **Character** | **人格・人間性**<br>誠実さ、責任感、リスペクト。<br>👉 *挨拶、道具の整理、時間を守る* |
| **5** | **Creativity** | **創造性**<br>自ら解決策を見つける力。<br>👉 *指示待ちではなく、自分で判断する* |

## 🔄 相互作用の図

これらは独立しているわけではなく、全て繋がっています。

> **Confidence (自信)** があるから ➔ **Creativity (創造性)** が発揮できる
> **Character (人格)** があるから ➔ **Connection (良い関係)** が築ける

## 📝 コーチの観察ポイント
私たちは、シュートが入ったかどうか（Competence）だけでなく、以下のようなシーンを評価します。

* ミスをした仲間に声をかけているか？ (**Connection**)
* 自分のマークマンが強い時、工夫して守ろうとしているか？ (**Creativity**)
* 審判や相手チームに敬意を払っているか？ (**Character**)
