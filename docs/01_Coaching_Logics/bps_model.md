# 3. BPSモデル (Bio-Psycho-Social)

選手を「バスケをしている機械」として見るのではなく、**「心と体を持った社会的な人間」**としてトータルで捉える、医学・心理学のアプローチです。

## 🏀 旭町ミニバスでの実践アクション

理論を具体的な行動に落とし込んでいます。

### 1. Bio（身体的アプローチ）
**「練習後のおにぎりプロジェクト」**
* **Why?** エネルギー不足（ガス欠）による成長阻害と、筋肉の分解を防ぐため。
* **Action:** 練習後30分以内に、おにぎり1個（タンパク質入り推奨）を食べる。
* **Sleep:** 小学生は9時間以上の睡眠が「仕事」です。

### 2. Psycho（心理的アプローチ）
**「教えすぎない指導（Guided Discovery）」**
* **Why?** 指示待ち人間にならず、自分でストレスや課題を乗り越える力をつけるため。
* **Action:** タイムアウトでは答えを言わず、「今、何が起きている？」と問いかける。

### 3. Social（社会的アプローチ）
**「保護者との連携（Parents as Partners）」**
* **Why?** コーチが見られない「家庭での時間（食事・睡眠）」こそが、選手の土台を作るため。
* **Action:** 「早く寝なさい」と声をかけることは、立派なコーチングであると伝える。
