# ライフスキルと人間形成 (Life Skills)

旭町MBCは「ダブル・ゴール」を目指します。
1. **選手の最大能力の達成:** バスケのスキルを最大限に伸ばす。
2. **人生の勝利:** バスケを通じて、立派な人間に成長すること。

## 1. 「2階建て」の理論 (The House Theory)

* **2階：心・技・体（On the Court）**
    * バスケのスキル、戦術、フィジカル。
* **1階：生 活 (Off the Court)**
    * 挨拶、返事、勉強、整理整頓、感謝。

### なぜ「生活」が1階なのか？
1階（生活）がグラグラの家は、どんなに立派な2階（バスケ）を載せても、プレッシャーや逆境という地震で簡単に倒壊してしまいます。

## 2. 自立と感謝
「自分の荷物は自分で持つ」「欠席連絡は自分でする」といった小さな自立の積み重ねが、22歳で自立した大人になるためのトレーニングです。
