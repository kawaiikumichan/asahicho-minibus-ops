# ライフスキルと人間形成 (Life Skills)

旭町MBCは「ダブル・ゴール（Double Goal）」を目指します。
1.  <strong>スコアボードの勝利:</strong> 試合に勝つこと。
2.  <strong>人生の勝利:</strong> バスケットボールを通じて、立派な人間に成長すること。

バスケットボールの上達には「心・技・体」が必要ですが、それら全てを支える<strong>4つ目の土台</strong>があります。
それが<strong>「生活（Life）」</strong>です。

## 1. 「2階建て」の理論 (The House Theory)

人間のパフォーマンスは「家」のようなものです。

### なぜ「生活」が大事なのか？

1階（生活）がグラグラの家は、どんなに立派な2階（バスケ）を載せても、ちょっとした地震（プレッシャーや逆境）で倒壊してしまいます。

* <strong>挨拶・返事:</strong> コミュニケーションの入り口。これができないと、パスも連携も生まれません。
* <strong>整理整頓:</strong> 「気づく力」を養います。荷物が乱れていることに気づけない選手は、ディフェンスのズレにも気づけません。
* <strong>勉強:</strong> 「考える力（脳の体力）」を鍛えます。判断力の土台になります。

## 2. オフコートの3大規律 (The 3 Disciplines)

「1階部分」を頑丈にするための具体的な行動ルールです。

### ① 挨拶と返事 (Greeting)
* <strong>ルール:</strong> 自分から先に、目を見て、明るい声でする。
* <strong>バスケとの繋がり:</strong>
    * 挨拶ができる子は、パスを呼ぶ声が出せます。
    * 返事ができる子は、コーチのアドバイスを素直に吸収し、成長スピードが速いです。

### ② 整理整頓 (Tidy up)
* <strong>ルール:</strong> バッシュを揃える。カバンを綺麗に並べる。ゴミを拾う。
* <strong>バスケとの繋がり:</strong>
    * 「乱れ」に気づく力は、ディフェンスの「ズレ」に気づく力と同じです。
    * 道具を大切にする選手は、ボールを大切にし、ターンオーバー（ミス）が減ります。

### ③ 時間管理 (Time Management)
* <strong>ルール:</strong> 集合時間の5分前には準備を完了する。
* <strong>バスケとの繋がり:</strong>
    * 時間の逆算ができる子は、試合終盤の残り時間に応じたプレー選択ができます。

## 4. 自立と感謝 (Autonomy & Gratitude)

「親やコーチに言われたからやる」のではなく、「自分で考えてやる」選手を育てます。

* <strong>自分の荷物は自分で:</strong> 重い遠征バッグも、水筒の準備も、極力親御さんは手伝わないでください。
* <strong>欠席連絡は自分で:</strong> 高学年になったら、休みの連絡は保護者経由ではなく、選手本人からキャプテンやコーチに連絡させます。
* <strong>感謝を言葉にする:</strong> 練習終わりには、迎えに来てくれた保護者に「ありがとう」と言葉で伝えます。
