# 試合の優先順位と方針

旭町MBCでは、ミニバス世代（U12）の最大の目的を「全選手の成長」と定義しています。
しかし、成長のためには「勝利を目指して真剣に取り組むプロセス」が不可欠です。

この相反する「育成」と「勝利」を両立させるため、試合の種類によって優先順位を明確に定めています。

## 優先順位の区分

| 試合種別 | 優先順位 | メンバー選定・起用方針 |
| :--- | :--- | :--- |
| **公式戦**<br>(県大会・市民大会など) | **勝利優先**<br> | **「勝ちにいく」メンバー選定を行います。**<br>出場時間は平等ではありません。勝利を目指す緊張感の中で、実力を発揮することを目指します。特に県大会などは勝利を最優先します。 |
| **練習試合**<br>(交流戦・リーグ戦など) | **育成優先**<br> | **「経験と挑戦」を重視した選定を行います。**<br>チーム全体の底上げを目的とし、失敗を恐れずにチャレンジする場とします。将来を見据えたメンバー構成を試すこともあります。 |

## 観戦・応援のルール

この方針を達成するため、保護者の皆様には以下のマナー遵守をお願いしています。

* **サイドコーチの禁止:** 育成の機会を奪うため、ベンチ以外からの指示（「パスだ！」「シュート！」など）は禁止です。
* **ポジティブな応援:** 自チームの選手を励ます声がけ（「ナイスプレー！」など）をお願いします。
