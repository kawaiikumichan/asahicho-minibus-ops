# 守備の基本原則 (Basic Defense)

旭町ミニバスは、JBAの**「マンツーマン推進ルール」**を遵守し、個々の守備能力（1対1で守り切る足）を鍛えます。
安易に引いて守る（ゾーン）のではなく、**「自分のマークマンに責任を持つ」** ことが大原則です。

## 🛡️ 3つの約束

### 1. オンボール (On Ball)
ボールを持っている相手には、厳しくプレッシャーをかけます。

<div style="background-color: #e3f2fd; border-left: 5px solid #2196f3; padding: 15px; margin-bottom: 20px;">
    <strong style="color: #1565c0; font-size: 1.1em;">① 激しいプレッシャー (Pressure)</strong>
    <p style="margin-top: 5px; color: #333;">
        腕一本分（ワンアーム）の距離まで近づき、自由にプレーさせません。
    </p>
    <ul style="margin-top: 5px; color: #0d47a1;">
        <li><strong>狙い:</strong> パスミスを誘う、ドリブルを止めさせる。</li>
        <li><strong>NG:</strong> 抜かれるのを怖がって離れすぎること。</li>
    </ul>
</div>

### 2. オフボール (Off Ball)
ボールを持っていないマークマンへの対応です。

<div style="background-color: #e3f2fd; border-left: 5px solid #2196f3; padding: 15px; margin-bottom: 20px;">
    <strong style="color: #1565c0; font-size: 1.1em;">② 正しいポジション (Positioning)</strong>
    <p style="margin-top: 5px; color: #333;">
        自分のマークマンだけを見るのはNGです。<br>
        <strong>「ボールとマークマンの両方が見える位置（ピストルスタンス）」</strong> を取ります。
    </p>
    <ul style="margin-top: 5px; color: #0d47a1;">
        <li><strong>ビジョン:</strong> 常に首を振って、ボールと人を確認する。</li>
        <li><strong>距離感:</strong> パスが出たらカットできる、ドライブされたら一歩助けられる位置。</li>
    </ul>
</div>

### 3. トランジション (Transition)
攻守の切り替えです。

<div style="background-color: #e3f2fd; border-left: 5px solid #2196f3; padding: 15px; margin-bottom: 20px;">
    <strong style="color: #1565c0; font-size: 1.1em;">③ 切り替えの速さ (Sprint Back)</strong>
    <p style="margin-top: 5px; color: #333;">
        シュートを打った瞬間、誰よりも速く自陣に戻ります。
    </p>
    <ul style="margin-top: 5px; color: #0d47a1;">
        <li><strong>ピックアップ:</strong> 戻りながら指を差して「俺は〇番！」と声を出し、自分のマークマンを捕まえる。</li>
    </ul>
</div>

<br>

<div style="border: 2px solid #d32f2f; background-color: #ffebee; padding: 15px; border-radius: 8px; color: #333;">
    <h3 style="margin-top: 0; color: #b71c1c;">⛔ ゾーン・ディフェンスの禁止</h3>
    <p>
        以下の守り方は「ゾーン（禁止事項）」とみなされるため行いません。<br>
        あくまで「個」を育てます。
    </p>
    <ul style="margin-top: 10px; font-weight: bold; color: #b71c1c;">
        <li>自分のマークマンが遠くにいるのに、ゴール下（ペイントエリア）に待ち構え続けること。</li>
        <li>マークマンの交代（スイッチ）を過度に行い、誰についてもいい状態で守ること。</li>
    </ul>
</div>
