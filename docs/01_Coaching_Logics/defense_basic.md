私たちは安易に引いて守る（ゾーン）のではなく、**「自分のマークマンに責任を持つ（個の力）」** を鍛えます。
その上で、単についていくだけではない **「意図を持って自ら仕掛ける（攻めるディフェンス）」** を基本方針とします。

## 🛡️ 3つの約束

### 1. オンボール (On Ball)
ボールを持っている相手には、プレッシャーをかけると同時に、**方向付け（ディレクション）**を行います。

<div style="background-color: #f1f8e9; border-left: 6px solid #4caf50; padding: 15px; margin-bottom: 20px;">
    <strong style="color: #2e7d32; font-size: 1.2em; display: block; margin-bottom: 10px;">① プレッシャー & ディレクション</strong>
    <p style="margin: 0 0 10px 0; color: #333;">
        ただ近づくだけでなく、<strong>「相手を苦手な方向へ追い込む」</strong>守り方をします。
    </p>
    <ul style="margin: 0; padding-left: 20px; color: #1b5e20;">
        <li style="margin-bottom: 5px;"><strong>激しいプレッシャー:</strong> 腕一本分（ワンアーム）まで近づき、自由を奪う。</li>
        <li style="margin-bottom: 5px;"><strong>ディレクション:</strong> 相手の進行方向を体で塞ぎ、行かせたい方向（サイドライン際や苦手な手の方）へ誘導する。</li>
        <li><strong>狙い:</strong> 相手に「やらされている」感覚を与え、ミスを誘発する。</li>
    </ul>
</div>

### 2. オフボール (Off Ball)
ボールを持っていないマークマンへの対応です。

<div style="background-color: #f1f8e9; border-left: 6px solid #4caf50; padding: 15px; margin-bottom: 20px;">
    <strong style="color: #2e7d32; font-size: 1.2em; display: block; margin-bottom: 10px;">② 正しいポジション (Positioning)</strong>
    <p style="margin: 0 0 10px 0; color: #333;">
        自分のマークマンだけを見るのはNGです。<br>
        <strong>「ボールとマークマンの両方が見える位置（ピストルスタンス）」</strong> を取ります。
    </p>
    <ul style="margin: 0; padding-left: 20px; color: #1b5e20;">
        <li style="margin-bottom: 5px;"><strong>ビジョン:</strong> 常に首を振って、ボールと人を確認する。</li>
        <li><strong>距離感:</strong> パスが出たらカットできる、ドライブされたら<strong>ヘルプ（カバー）</strong>に行ける位置。</li>
    </ul>
</div>

### 3. トランジション (Transition)
攻守の切り替えです。

<div style="background-color: #f1f8e9; border-left: 6px solid #4caf50; padding: 15px; margin-bottom: 20px;">
    <strong style="color: #2e7d32; font-size: 1.2em; display: block; margin-bottom: 10px;">③ 切り替えの速さ (Sprint Back)</strong>
    <p style="margin: 0 0 10px 0; color: #333;">
        シュートを打った瞬間、誰よりも速く自陣に戻ります。
    </p>
    <ul style="margin: 0; padding-left: 20px; color: #1b5e20;">
        <li><strong>ピックアップ:</strong> 戻りながら指を差して「俺は〇番！」と声を出し、自分のマークマンを捕まえる。</li>
    </ul>
</div>

        <li>マークマンの交代（スイッチ）を過度に行い、誰についてもいい状態で守ること。</li>
