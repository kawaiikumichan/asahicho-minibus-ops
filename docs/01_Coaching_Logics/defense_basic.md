# 守備の基本原則 (Basic Defense)

旭町ミニバスは、JBAの**「マンツーマン推進ルール」**を遵守し、個々の守備能力（1対1で守り切る足）を鍛えます。
安易に引いて守る（ゾーン）のではなく、**「自分のマークマンに責任を持つ」** ことが大原則です。

## 🛡️ 3つの約束

### 1. オンボール：激しいプレッシャー (Pressure)
ボールを持っている相手には、腕一本分（ワンアーム）の距離まで近づき、自由にプレーさせません。
* **狙い:** パスミスを誘う、ドリブルを止めさせる。
* **NG:** 抜かれるのを怖がって離れすぎること。

### 2. オフボール：ポジション取り (Positioning)
ボールを持っていないマークマンに対しては、**「ボールとマークマンの両方が見える位置（ピストルスタンス）」** を取ります。
* **ビジョン:** 常に首を振って、ボールと人を確認する。
* **距離感:** パスが出たらカットできる、ドライブされたら一歩助けられる位置。ただし、下がりすぎてペイントエリアに常駐することは禁止（ゾーン判定防止）。

### 3. トランジション：切り替えの速さ (Transition)
シュートを打った瞬間、オフェンスからディフェンスへ即座に切り替えます。
* **Sprint Back:** 誰よりも速く自陣に戻る。
* **Pick Up:** 戻りながら、指を差して「俺は〇番！」と声を出し、自分のマークマンを捕まえる。
