# 夢の設計図：22歳への航海図 (Dream Blueprint)

旭町MBCが目指すゴールは、12歳の結果ではなく、**「22歳〜25歳の君」** が社会で輝いていることです。

## 1. 夢と「今」をつなぐ方法
子供たちの夢（22歳）と、今の行動が離れている（乖離している）時、以下の視点で対話を行い、一本の線につなげます。

### ① 未来の自分に相談する（バックキャスティング）
「22歳の理想の君なら、今の練習でどんな準備をしていると思う？」と問いかけ、未来の自分を「今の自分を導く先生」にします。

### ② 「あり方（Being）」を深掘りする
「プロになりたい（Doing）」という夢の裏にある、「みんなを驚かせたい」「仲間に信頼されたい」という **「どんな人でありたいか（Being）」** を共有します。これがあれば、目標が変わっても軸はブレません。

### ③ 中間目標（マイルストーン）の設定
22歳までの間に、「15歳（中学卒業）」「18歳（高校卒業）」のチェックポイントを設けます。

## 2. ワークシートの構成（イメージ）

| ステージ | 年齢 | 目標・なりたい姿（Being & Doing） |
| :--- | :---: | :--- |
| **【ゴール】** | **22-25歳** | **自立した社会人として活躍する** |
| 中間点 | 18歳 | （例）高校で全国大会を目指す / 文武両道 |
| 中間点 | 15歳 | （例）県選抜に選ばれる / 基礎を極める |
| **【通過点】** | **12歳** | **（個人の目標を自由に書く）** |

---
## 💡 挫折は「ルート再検索」
もし途中の目標に届かなかったとしても、それは失敗ではありません。夢の設計図があれば、22歳のゴールへ続く「新しい道」をすぐに見つけることができます。
