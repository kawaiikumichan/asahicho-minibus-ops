# シュート・スタンダード（ワンハンドの推奨）

旭町MBCでは、世界標準の技術習得を目指し、女子選手であっても **ワンハンドシュート** の習得を推奨しています。

## なぜワンハンドなのか？

ボースハンド（両手打ち）には「力が弱い時期でもボールを遠くに飛ばしやすい」というメリットはありますが、学年が上がりプレーのレベルが高くなるにつれて、以下の**致命的な制限**が発生します。

### 🚫 ボースハンドのデメリット（具体的な制限）

**1. 「動き」の制限：止まらないと狙った場所に飛ばない**
* ボースハンドは、**両手でボールを挟んでコントロール**します。
* そのため、走りながら（ランニングシュート）や横に動きながら打つと、体のブレがそのまま手のブレに繋がり、左右の力のバランスが崩れてしまいます。
* 結果として、**動きながら打つと、思っている方向にボールが飛びません。**
* 正確に決めるためには、どんな時でも**一度完全に止まって（ストップして）**、体を安定させる必要が出てきてしまいます。

**2. 「プレーの幅」の制限：多彩なフィニッシュができない**
* 上記の理由から、セットシュート（止まって打つシュート）には向いていますが、**動きの中で打つシュート**への応用が困難です。
* 現代バスケに必須のフローターシュートや、ディフェンスをずらして打つステップバックシュートなどが、構造上極めて難しくなります。

### ✅ ワンハンドのメリット（推奨する理由）

これらを解消し、自由なプレーを可能にするのがワンハンドシュートです。

**1. 動きの中でコントロールできる**
* 片手でボールの中心を捉えるため、体が流れていたり、斜めを向いていたりしても、**腕の振りだけで方向をコントロール**できます。
* これにより、ドリブルの勢いを殺さずにスムーズにシュートへ移行できます。

**2. ステップワークとの相性が良い**
* 縦・横・斜めへのステップと連動させやすく、相手をかわしながらのシュートが容易になります。
* ボールの持ち替えがスムーズで、パスやドリブルへの切り替えも速くなります。

**3. 世界標準・適期教育**
* 男女問わず、現代バスケットボールのスタンダードです。
* 神経系が最も発達するU12世代（ゴールデンエイジ）で、この「一生使えるフォーム」を身につけるべきです。
