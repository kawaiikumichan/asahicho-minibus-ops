# 戦術：5Out Motion Offense

旭町ミニバスでは、特定のセットオフェンスではなく、**5Out（ファイブ・アウト）**を採用しています。

## 1. 導入の目的
* **全員に1on1の機会を作る**
    * 中を空ける（インサイドに誰もいない）ことで、ドライブするための広大なスペースを作ります。
* **ポジションレスな育成**
    * 「背が高いからゴール下」と決めつけず、全員がリングに向かってドライブし、パスを出し、シュートを打つ技術を磨きます。
* **判断力を養う**
    * 「決められた動き」ではなく、「味方がドライブしたら合わせる」「パスしたら走る」という**原則（Principle）**に基づいて、選手自身が判断して動きます。

## 2. 基本原則（ルール）
### Rule 1: Pass & Cut（パス＆カット）
* パスを出したら、必ずその場から動く。
* 基本は「バスケットへ切れ込む（バスケット・カット）」こと。

### Rule 2: Fill the Spot（スペースを埋める）
* 味方がカットして空いた場所（スポット）を、隣の選手が素早く埋める。
* リング状にぐるぐると人が動き続けるイメージ。

### Rule 3: Attack the Gap（ギャップを攻める）
* ボールを持ったら、まずはリングを見る。
* ディフェンスのズレ（ギャップ）があれば、迷わず1on1を仕掛ける。

## 3. コーチングのポイント
* **動きを止めない:** 「止まっている時間が長い」ことが最大のエラーです。
* **ミスを責めない:** 1on1で抜かれても、シュートを外してもOK。ただし「攻める姿勢がなかった」時だけ指導します。
