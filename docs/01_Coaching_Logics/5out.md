# 攻撃戦術：5アウト・オフェンス (5-Out Motion Offense)

旭町MBCでは、チーム戦術として **「5アウト・オフェンス」** を通年で採用しています。
特定のセットオフェンス（完全に決められた動き）だけではなく、原則に基づいて全員が動く「モーションオフェンス」です。

## 1. 導入の目的

育成年代（U12）に最適なシステムとして、以下の理由から採用しています。

* **全員参加のオフェンス (Total Offense)**
    * 特定のセンター（インサイドプレーヤー）を作らず、5人全員がアウトサイドのスペースを使います。
    * 「背が高いからゴール下」という固定観念をなくし、全員がボールに触れ、攻める機会を作ります。
* **オフボール・スキルの向上**
    * ボールを持っていない時（オフボール）にどう動くかが重要になる戦術です。
    * 味方のドライブに合わせて動く、スペースを埋めるなど、バスケットボールIQを高めます。
* **個の強さを引き出す**
    * 中を空ける（インサイドに誰もいない）ことで、1on1を仕掛けるための広大なスペースが生まれます。
    * 意図的に「1on1の強化」が必須となる環境を作ることで、個人の突破力を磨きます。

## 2. 採用しているフォーメーション

旭町MBCでは、基本の5アウトに加え、以下のパターンを状況に応じて使用します。

* **5アウト・8オフェンス (Figure 8)**
    * 基本的な動きのパターン。8の字を描くようにパスとランを繰り返します。
* **1-4 フォーメーション**
    * ガード1人がトップ、他の4人が横一列（フリースローラインの高さ）に並ぶ配置です。

## 3. 基本原則（3つのルール）

選手は「ロボットのように動く」のではなく、以下の原則に基づいて**自分で判断**して動きます。

### Rule 1: Pass & Cut（パス＆カット）
* パスを出したら、その場に留まらない。
* 必ず**「バスケットへ切れ込む（バスケット・カット）」**動きを行い、ディフェンスを脅かします。

### Rule 2: Fill the Spot（スペースを埋める）
* 味方がカットして空いた場所（スポット）を、隣の選手が素早く埋めます。
* リング状にぐるぐると人が動き続け、常に攻めるスペースを確保します。

### Rule 3: Attack the Gap（ギャップを攻める）
* ボールを持ったら、まずはリングを見る（Look at the Ring）。
* ディフェンスのズレ（ギャップ）があれば、パス回しを中断して**迷わず1on1を仕掛けます**。

## 4. コーチングのポイント

* **動きを止めない:** 「止まっている時間が長い」ことが最大のエラーです。
* **ミスを責めない:** 1on1で抜かれても、シュートを外してもOKです。
* **判断を評価する:** 「攻める姿勢（GUN）」があったか、適切なスペースに動けたかという「プロセス」を指導します。
