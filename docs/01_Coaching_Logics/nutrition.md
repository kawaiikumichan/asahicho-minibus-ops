# 4. 体づくりと食育 (Body & Nutrition)

世界標準の育成（BPSモデルの **"Bio"**）に基づき、練習と同じくらい「食事と休息」を重視します。

## 🍙 「練習後のおにぎり」プロジェクト

旭町ミニバスでは、**練習が終わった直後（30分以内）の補食** を推奨しています。

### なぜ今、食べるのか？
帰宅して夕食を食べるまでの「空白の1〜2時間」が、成長の分かれ道です。

### 具体的なアクション
* **タイミング:** 練習直後（着替えや片付けの合間、帰り道）。
* **量:** コンビニおにぎり1個、または小さめのおにぎり。
* **具材:** 鮭、ツナ、鶏そぼろ（タンパク質も一緒に摂れるため最強です）。

## 💤 睡眠と休養 (Rest)

「寝るのも練習」です。夜更かしをして練習に来ても、パフォーマンスは上がらず、怪我のリスクだけが上がります。

### 推奨される睡眠時間
* **小学生アスリート： 9時間以上**
* 成長ホルモンは、深い眠りの間に最も分泌されます。

### 成長痛（オスグッド等）への対応
* 膝や踵が痛いときは、勇気を持って休ませてください。
* 痛みをかばってプレーすると、フォームが崩れ、他の場所（腰など）を痛める原因になります。
