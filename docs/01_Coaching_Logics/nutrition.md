# 食育と水分補給 (Nutrition & Hydration)

旭町MBCでは、練習と同じくらい **「食べる・飲む」** ことを重要視しています。
ガス欠（エネルギー切れ）や脱水状態では、いくら良い練習をしても身になりません。

## 1. 水分補給のルール (Hydration Strategy)

「喉が渇いた」と感じた時には、すでに体は脱水状態（パフォーマンス低下）が始まっています。
**「喉が渇く前に飲む」** をルール化します。

### 💧 飲むタイミングと量
がぶ飲みは胃に負担をかけるため、**「こまめに少量ずつ」** が鉄則です。

| タイミング | 量の目安 | 目的 |
| :--- | :--- | :--- |
| **練習前** | コップ1杯 (200ml) | あらかじめ体内のタンクを満たしておく。 |
| **練習中** | 15〜20分ごとに一口 | 汗で失った分を即座に補う。<br>※コーチが強制的に給水タイムを設けます。 |
| **練習後** | 失った体重分 | リカバリー（回復）のため。 |

### 🥤 飲み物の種類
* **基本:** **麦茶** または **水**。
    * ミネラルが含まれる麦茶は推奨されます。
* **激しい練習中:** **スポーツドリンク**（ハイポトニック推奨）。
    * 糖分が多すぎるジュース類（炭酸飲料など）は、血糖値が急激に乱れるため練習中は禁止です。

### ✅ セルフチェック：おしっこの色
トイレに行った際、自分のおしっこの色を見て判断させます。
* **透明〜薄い黄色:** OK（足りている）
* **濃い黄色〜茶色:** **危険（脱水状態）** → すぐに水を飲むこと！

## 2. 補食とおにぎり (Energy Recovery)

激しく動いた後の体は、スポンジのように栄養を吸収します。
この **「ゴールデンタイム（練習後30分以内）」** を逃さないことが、強い体を作る鍵です。

### 🍙 「練習後のおにぎり」の推奨
旭町MBCでは、練習後に素早くエネルギーを補給するため、おにぎりの持参を推奨しています。

* **内容:** 具材は何でもOKですが、消化の良いもの（鮭、梅など）がおすすめです。
* **タイミング:** 帰宅してから夕食を食べるまで待つと時間が空きすぎるため、**練習が終わって着替えたらすぐ（帰り道や車内）** に食べさせます。

### 🍊 100%オレンジジュース
疲労回復には、糖質と一緒に **クエン酸** や **ビタミンC** を摂ると効果的です。
おにぎりとセットで、100%オレンジジュースを飲むのもプロ選手が実践する優れたリカバリー法です。

## 3. 試合当日の食事 (Game Day)

試合で最高のパフォーマンスを出すための食事スケジュールです。
**「消化の時間」** を逆算して食べます。

* **試合開始 3時間前:** **食事完了**
    * ご飯、うどん、餅などの炭水化物をしっかり摂る。
    * 油っこいもの（揚げ物、ラーメン）や生ものは避ける。
* **試合開始 1時間前:** **エネルギーゼリー・バナナ**
    * すぐにエネルギーに変わる固形物でないものを軽く摂る。
* **試合直前・ハーフタイム:** **水分・ブドウ糖**
    * スポーツドリンク等で血糖値を維持する。
