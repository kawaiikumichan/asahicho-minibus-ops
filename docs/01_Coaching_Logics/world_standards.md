# 世界標準の育成モデルと旭町ミニバス

私たちの指導方針は、世界的な育成トレンドに基づいています。

## 1. LTADモデル (長期競技者育成)
カナダ発、JBAも推奨するモデルです。
* **U12は「FUNdamentals」の時期**
    * 勝利よりも「楽しさ」と「多面的な運動スキル」を優先します。
    * 早期専門化（ポジション固定やパターン戦術）は、将来の伸びしろを奪うため行いません。

## 2. The 5 Cs (優れた選手の定義)
英国サッカー等で採用される指標です。技術だけでなく人間性を重視します。
1. **Competence (競技能力)**: スキル・戦術眼
2. **Confidence (自信)**: 失敗を恐れない心
3. **Connection (つながり)**: 仲間・コーチとの信頼
4. **Character (人間性)**: 誠実さ・責任感
5. **Creativity (創造性)**: 自ら解決する力

## 3. BPSモデル (Bio-Psycho-Social)
選手を3つの側面からトータルで支えます。
* **Bio (身体)**: 食事（練習後のおにぎり）、睡眠、成長痛ケア
* **Psycho (心理)**: モチベーション、ストレス管理（問いかける指導）
* **Social (社会)**: 学業、友人、家庭とのバランス
