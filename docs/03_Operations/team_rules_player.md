# 選手への約束 (Rules for Players)

旭町MBCの選手として守ってほしいお約束（ルール）です。
これらは、チーム全員が楽しく安全にバスケットボールをするため、そして選手一人ひとりが成長するために大切なことです。

| お約束（具体的な行動） | なぜ大切なの？（理由） |
| :--- | :--- |
| [cite_start]**挨拶は自分から元気な声で**しましょう。<br>（コーチ、メンバー、保護者、対戦相手へ） [cite: 622] | 自分から心を開いてコミュニケーションをとるためです。 |
| [cite_start]**返事は相手に伝わる大きい声で**しましょう。<br>（周りの声を待たず、自分のタイミングでする） [cite: 622] | 「わかりました」「きこえました」という自分の意思を相手にしっかり伝えるためです。 |
| [cite_start]掃除や片付けは**自分から進んで**やりましょう。 [cite: 622] | チームの一員として、誰かに言われる前に行動できる人になるためです。 |

## 2. 連絡・相談

| お約束（具体的な行動） | なぜ大切なの？（理由） |
| :--- | :--- |
| [cite_start]**お休みする際は事前に**コーチとキャプテンに伝えましょう。<br>（キャプテンのノートにも書いてください） [cite: 622] | チームの練習計画や準備に協力し、約束（規律）を守るためです。 |

## 3. 身支度・荷物管理

| お約束（具体的な行動） | なぜ大切なの？（理由） |
| :--- | :--- |
| [cite_start]持ち物に名前を書き、脱いだものをすぐにバッグに入れましょう。<br>**荷物をひとつにきれいに並べましょう**。 [cite: 622] | 自分の道具を大切にし、自分で管理する力（責任感）をつけるためです。 |

## 4. 車・会場でのマナー

| お約束（具体的な行動） | なぜ大切なの？（理由） |
| :--- | :--- |
| [cite_start]「お願いします」「ありがとうございました」を<br>**運転手に聞こえるように伝えましょう**。 [cite: 622] | 送迎して支えてくれる人への「感謝の気持ち」を言葉で伝えるためです。 |
