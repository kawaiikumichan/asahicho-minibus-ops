# 車出しのお願いと安全ガイド (Transportation Support)

旭町MBCの活動は、保護者の皆様の温かいサポート（車出し）によって支えられています。
遠征に行けるのも、試合ができるのも、ハンドルを握ってくださる方々のおかげです。心より感謝申し上げます。

このガイドラインは、**「善意で協力してくださるドライバーの皆様を守るため」** に定めています。
安心してご協力いただけるよう、必ずご一読ください。

## 1. 保険についての確認 (Insurance Check)

新たに特別な保険に加入していただく必要はありません。
現在ご加入中の**「一般的な自動車保険（任意保険）」**で対応可能です。
念のため、車出しをする前にご自身の保険内容をご確認ください。

* <strong>✅ 確認していただきたいこと:</strong>
    * <strong>対人・対物賠償:</strong> 「無制限」になっているか。
    * <strong>同乗者への補償:</strong> 他人の子供を乗せた場合も補償されるか。（人身傷害補償など）
    * ※ボランティア活動（無償）での送迎ですので、通常の「日常・レジャー使用」の範囲内で補償されます。
* <strong>🛡️ チームが加入している保険（スポーツ安全保険）:</strong>
    * チーム全員が加入する「スポーツ安全保険」は、移動中の事故による**「選手の怪我」**を補償します（治療費や入院費）。
    * ただし、<strong>「車の修理費」や「相手への賠償」はカバーされません。</strong> そのため、ドライバーご自身の任意保険による保護が必要となります。

## 2. プレッシャーを感じない運転を (No Pressure)

皆様にお願いしたいのは、**「子供たちを無事に送り届けること」** だけです。
時間に追われる必要はありません。

* <strong>「遅れてもOK」の原則:</strong>
    * 渋滞や道の間違いで遅刻しても、誰も責めません。
    * <strong>「急ぐこと」よりも「無事に着くこと」が100倍重要です。</strong> 遅れそうな時は、同乗している他の保護者や子供に連絡させれば大丈夫です。
* <strong>定員とシートベルト:</strong>
    * 無理な乗車は事故のもとです。ゆとりを持った配車を組みますので、ご安心ください。
    * 後部座席の子供たちにも、必ずシートベルトを締めさせてください（それが完了するまで発進しなくて構いません）。

## 3. 子供たちへの指導 (Passenger Manners)

車出しは「タクシー」ではありません。
ドライバーが気持ちよく運転できるよう、チームとして子供たちに以下のマナーを徹底させています。
もし守れていない子がいたら、<strong>遠慮なく叱るか、コーチに報告してください。</strong>

* <strong>車内は「動く部屋」:</strong>
    * 騒いだり、暴れたりすることは、運転手の集中を削ぐ危険行為です。絶対にさせません。
* <strong>感謝の気持ち:</strong>
    * 乗降時の「お願いします」「ありがとうございました」の挨拶を徹底します。
* <strong>汚れ防止:</strong>
    * 泥のついたバッシュやカバンへの配慮など、各家庭の車のルールに従わせます。

## 4. もしもの時のサポート (In Case of Accident)

万が一、事故に遭われた場合も、一人で抱え込む必要はありません。
まずは身の安全を確保し、落ち着いて以下の手順で頼ってください。

> 互いに感謝し合い、気持ちの良い遠征にしましょう。
