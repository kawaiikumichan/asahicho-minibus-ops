# 保護者へのお願い (Requests to Parents)

旭町MBCは、保護者の皆様のご協力によって運営されています。
「FUN & GUN」の理念のもと、子供たちが安全にバスケットボールに打ち込める環境を作るため、以下のルールとお願いへのご理解・ご協力をお願いいたします。

## 1. 連絡・情報共有について

円滑な運営のため、以下のルールを定めています。

| 項目 | 内容 |
| :--- | :--- |
| **連絡ツール** | **LINE**（グループ参加）および **TeamHub**（アプリ）の利用を必須とします。<br>返信が必要なものは期限内にお願いします。 |
| **欠席連絡** | 練習や試合を休む際は、事前に**TeamHub/LINEでの報告**をお願いします。<br>（メンバーからコーチへの報告と、キャプテンへの連絡を行ってください） |
| **緊急連絡** | 練習・試合中は、怪我や体調不良等の連絡が取れるようにご配慮ください。 |

## 2. 身支度・持ち物について

安全管理のため、ご家庭での指導をお願いします。

* **服装**: フード付きの衣服は、練習中の怪我予防のため着用禁止です。髪が長い場合はゴムで結んでください。金属製のピンや装飾は禁止です。
* **持ち物**: 練習に不要なものは持たせないでください。
    * **スマホ・GPS**: GPS用途での所持は可能です。ただし、最低限の連絡以外は利用しないよう指導をお願いします。
    * **紛失・破損**: トラブルについてチームは一切の責任を負いません。

## 3. 送迎・車出しについて

移動の安全確保と、保護者の負担軽減を両立させるためのルールです。

* **送迎の責任**: 練習会場や集合場所への移動は、各ご家庭の責任にて行ってください。
    * 安全確保のため、保護者による送迎を推奨します。
* **車出し協力**: 遠征時の配車（車出し）に積極的なご協力をお願いします。
    * **片道のみOK**: 負担軽減のため、往路・復路いずれかのみの協力も可能です。
    * **燃料費**: 車出しをする方が損をしないよう、毎月設定される単価（少し多め）で計算しお支払いします。
    * **高速道路**: 30分程度の時間短縮が見込める場合に利用します（要事前確認）。
* **安全管理**: 全席**シートベルトの着用**を徹底してください。

## 4. 当番・会場活動への協力

子供たちだけでは難しい環境整備をサポートします。

| 役割 | 内容 |
| :--- | :--- |
| **練習見守り** | 全体練習時は、当番制で選手の見守りを行います。<br>（準備・片付け、怪我時の応急処置、水筒補充など） |
| **会場準備** | 窓開け、ゴールの設置、カーテン開閉など。<br>※子供のみでは対応が難しい作業にご協力ください。 |
| **会場片付け** | 消毒補助、トイレ掃除、戸締まり、ゴールの撤去など。 |

## 5. 試合観戦・応援のマナー

選手の成長を守るための観戦ルールです。

* **指示出しの禁止**: 観戦席から選手へ指示を出すこと（「打て！」「守れ！」「右だ！」など）はルールで禁止されています。
    * これらはテクニカルファウルの対象となります。
* **審判へのリスペクト**: 審判へのクレームや野次は厳禁です。
* **推奨される応援**: 自チームの選手を応援する声かけをお願いします。
    * 相手チームのミスに対する声かけは禁止です。
* **服装**: 試合の際はチームTシャツ（青色）・トレーナーの着用をお願いします。

---
皆様のご協力のおかげで、子供たちはバスケットボールを全力で楽しむことができます。
