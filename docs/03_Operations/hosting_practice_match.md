# ホーム練習試合の手引き (Hosting Practice Games)

旭町MBCがホーム（自校）に他チームを招いて行う、練習試合の運営ガイドです。
公式戦とは異なり、比較的リラックスした雰囲気で行われますが、**「おもてなしの心」** と **「安全への責任」** は変わりません。

## 1. ホーム開催のメリット (Why Host?)

準備は大変ですが、それを上回るメリットが子供たちにあります。

* <strong>慣れた環境:</strong> いつものリング、床でプレーできるため、練習の成果を発揮しやすい。
* <strong>移動負担なし:</strong> 車酔いや早起きの負担がなく、体調万全で臨める。
* <strong>全員出場:</strong> 遠征人数に縛られず、低学年や初心者も含めて全員が試合経験を積める（B戦・サテライト戦が組みやすい）。

## 2. 当日の役割分担 (Roles)

集合は通常 **開場の1時間前** です。コーチと保護者が協力して設営します。

### 🏟️ 会場設営（全員で）
* <strong>ゴール設置:</strong> ミニバスの高さは **260cm** です。必ずメジャーで計測します。
* <strong>ラインテープ:</strong> マイクロバスケットボール（低学年）用など、必要に応じてラインを貼ります。
* <strong>ベンチ設営:</strong> 選手用の椅子、TO席（デジタイマー設置）を作ります。

### 🏥 安全管理（ホストの責任）
* <strong>AEDの確認:</strong> 体育館のどこにあるか（鍵は開いているか）を確認します。
* <strong>病院情報の把握:</strong>
    * 万が一の怪我に備え、<strong>「今日の当番医（救急病院）」</strong> をスマホ等ですぐ出せるようにしておきます。
    * 相手チームから聞かれたら即答できるようにしましょう。

### 🚗 駐車場係（男性保護者推奨）
* <strong>誘導:</strong> 事前に伝えた台数をチェックし、指定場所に隙間なく誘導します。
* <strong>巡回:</strong> 近隣（路上・コンビニ）への不正駐車がないか確認します。

### 🍵 受付・接待
* <strong>お迎え:</strong> 玄関で「おはようございます！」と明るく迎えます。
* <strong>案内:</strong> トイレ、更衣室、荷物置き場を案内します。
* <strong>お茶出し:</strong> 相手チームのコーチにペットボトルのお茶やお菓子（茶菓子）を出します（※チーム規定による）。

## 3. 撤収と片付け
* <strong>モップ掛け:</strong> 感謝を込めて、フロア全体を綺麗にします。
* <strong>ゴミ確認:</strong> 相手チームのベンチ付近にゴミが残っていないか確認し、あればこちらで処分します。

> <strong>ポイント</strong>
> 練習試合は「子供たちの修行の場」です。
> 運営も完璧を目指しすぎず、<strong>「安全に試合ができればOK」</strong> くらいの気持ちで協力し合いましょう。
