# 運営体制と役割分担 (Organization)

旭町MBCは、コーチスタッフと保護者の協力体制（役割分担）によって運営されています。
スローガンは **「業務集中から分担へ」** です。特定の個人に負担が偏らない、持続可能なチーム運営を目指しています。

## 1. コーチングスタッフ (Coaching Staff)

| 役職 | 主な役割・責任 |
| :--- | :--- |
| **ヘッドコーチ (HC)**<br>兼 チーム代表 | **【チームの統括】**<br>・女子チームの指導方針決定、戦術構築<br>・公式戦での采配、メンバー選定<br>・対外試合のスケジュール調整 |
| **アシスタントコーチ (AC)** | **【指導・補佐】**<br>・ヘッドコーチの補佐、練習指導<br>・公式戦・練習試合でのベンチワーク<br>・審判対応（要ライセンス取得） |
| **サポートコーチ** | **【練習サポート】**<br>・練習時の指導補助、選手の集中力維持のサポート<br>・対人練習の相手（ディフェンス役など）<br>※未経験者や保護者コーチも歓迎しています。 |

## 2. 運営・マネジメント (Management)

保護者の役員および係の役割です。

| 役職 | 主な役割 |
| :--- | :--- |
| **運営役員**<br>(事務・会計など) | ・対外的な窓口業務<br>・会計管理（部費・経費の管理・集金）<br>・Tシャツや備品の管理・発注<br>・連絡ツール（TeamHub）の管理・更新 |
| **マネージャー**<br>(お世話係) | ・女子チームのマネジメント全般<br>・練習・試合での引率、怪我や体調不良時の対応<br>・バスケスキル以外の指導（挨拶、靴並べ、整理整頓など）<br>・体育館やイベント会場の予約・鍵管理 |

## 3. 保護者サポーター (Parents Support)

役員以外の保護者全員で分担して行う役割です。「できる人が、できる時に」協力します。

| 係・役割 | 内容 |
| :--- | :--- |
| **広報・SNS係** | ・WebサイトやInstagramの投稿・管理<br>・「検索されやすい旭町」を目指すSEO対策 |
| **撮影係** | ・試合やイベントの写真・ビデオ撮影<br>・撮影データの共有・管理（Googleフォト等） |
| **配車（車出し）** | ・遠征時の選手輸送<br>・片道のみの協力も可能（燃料費精算あり） |
| **当番（見守り）** | ・練習時の鍵開け、施錠<br>・練習中の見守り（体調不良者への対応など） |
