# 練習メニューの作り方 (Practice Planning)

良い練習には、必ず <strong>「今日のテーマ（ねらい）」</strong> があります。
漫然といつものメニューを繰り返すのではなく、<strong>「今日、選手に何を持ち帰らせたいか」</strong> を設計してからコートに立ちましょう。

## 1. 練習計画の 3ステップ

練習を作る際は、以下の手順で考えます。

### Step 1: テーマを決める (One Day, One Theme)
欲張ってはいけません。1回の練習で伝えられることは1つだけです。

<div style="background-color: #f1f8e9; border-left: 5px solid #8bc34a; padding: 15px; margin-bottom: 20px;">
    <strong>✅ テーマの良い例</strong>
    <ul>
        <li>「今日は、ボールをもらう前の準備（ミート）を徹底する」</li>
        <li>「今日は、ディフェンスの方向付け（ディレクション）だけを意識する」</li>
    </ul>
    <br>
    <strong>❌ テーマの悪い例</strong>
    <ul>
        <li>「シュートもパスもディフェンスも全部やる」<br>→ 焦点がボケて、結局何も身につきません。</li>
    </ul>
</div>

### Step 2: 構成を組む (Structure)
決めたテーマを、段階を追って習得できるように並べます。

| 順序 | パート | 内容・参照 | ポイント |
| :--- | :--- | :--- | :--- |
| <strong>1</strong> | <strong>導入 (15分)</strong> | <strong>コーディネーション</strong> | 脳と体を温める。テーマを伝える。 |
| <strong>2</strong> | <strong>分解 (30分)</strong> | <strong>ドリルバンク</strong> | テーマとなる技術を、DFなし〜軽い負荷で練習する。 |
| <strong>3</strong> | <strong>実戦 (30分)</strong> | <strong>3対3 / 5アウト</strong> | ディフェンスがいる状態で、その技術が使えるか試す。 |
| <strong>4</strong> | <strong>確認 (15分)</strong> | <strong>スクリム (5対5)</strong> | ゲーム形式。コーチは止めすぎない。 |

### Step 3: 会場とスタッフの配置 (Venue & Staff)
旭町MBCでは、Aチームと育成チームが別会場で練習することが多くあります。
それぞれの会場責任者（リードコーチ）が、以下の役割を果たします。

| 会場 | リードコーチ | メニュー方針 | 注意点 |
| :--- | :--- | :--- | :--- |
| <strong>Aチーム会場</strong> | <strong>ヘッドコーチ (HC)</strong> | <strong>強度と戦術</strong><br>試合に向けた修正と、チームルールの徹底。 | 選手のコンディション（怪我）の見極め。 |
| <strong>育成チーム会場</strong> | <strong>AC / サポートコーチ</strong> | <strong>基礎と楽しさ</strong><br>個のスキルアップと、バスケを楽しむ空気作り。 | <strong>安全管理の徹底。</strong><br>HC不在時こそ、挨拶や整理整頓（ライフスキル）が緩まないようにする。 |

## 2. 実際のメニュー構成例 (90分〜120分)

会場ごとにテーマ設定を変えることが一般的です。

### 🅰️ Aチーム会場（テーマ：5アウトの「合わせ」）

| 時間 | カテゴリ | メニュー名 | コーチの声掛け（意識） |
| :--- | :--- | :--- | :--- |
| <strong>15分</strong> | <strong>Coordination</strong> | テニスボールドリブル | 「顔を上げて周りを見よう（空間認知）」 |
| <strong>10分</strong> | <strong>Run</strong> | 3メン（切り替え） | 「パスを出したら止まらずに走れ！」 |
| <strong>30分</strong> | <strong>Tactics</strong> | 3対3 (ギャップアタック) | 「ズレができたら攻める！止まったらパス！」 |
| <strong>30分</strong> | <strong>Game</strong> | 5対5 スクリム | 「ミスしてもいいから、さっきの合わせを使ってみよう」 |

### 🐣 育成チーム会場（テーマ：ドリブルを強くする）

| 時間 | カテゴリ | メニュー名 | コーチの声掛け（意識） |
| :--- | :--- | :--- | :--- |
| <strong>20分</strong> | <strong>Fun</strong> | 鬼ごっこ / だるまさんが転んだ | 「パワーポジションで止まれているかな？」 |
| <strong>30分</strong> | <strong>Skill</strong> | ハンドリング / 1対1 | 「ボールを強く突こう！失敗してもOK！」 |
| <strong>30分</strong> | <strong>Game</strong> | ミニゲーム (3対3など) | 「どんどんシュートを打ってみよう！」 |

## 3. コーチング・ティップス (Tips)

* <strong>「並ばせる」のをやめよう:</strong>
    * 10人が1つのゴールに並ぶと、1人あたり数回しか打てません。
    * 2グループに分ける、サイドのリングを使う、待ち時間にハンドリングをさせるなどして、<strong>「常に全員が動いている状態」</strong> を作りましょう。
* <strong>説明は「短く」:</strong>
    * 小学生の集中力は長く続きません。
    * 説明は30秒〜1分以内で。やってみて、止めて、修正するほうが伝わります。
* <strong>PDCAを回す:</strong>
    * 練習後、「今日のテーマは達成できたか？」をコーチ間で話し合い（LINE等）、次回のメニューに反映させます。
