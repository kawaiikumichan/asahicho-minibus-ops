    <h1 style="color: #d84315; border-bottom: 3px solid #d84315; padding-bottom: 10px;">３．（５） シュート基礎：ワンハンドの型</h1>

    <div style="background-color: #fbe9e7; padding: 20px; border-radius: 10px; margin-bottom: 30px; border-left: 5px solid #d84315;">
        <p style="margin-top: 0; font-weight: bold; color: #bf360c;">🏀 指導のコア・コンセプト</p>
        ワンハンドシュートは、腕の力ではなく<strong>「全身の連動（キネティックチェーン）」</strong>で飛ばします。<br>
        足元から指先へエネルギーを伝える「５つのフェーズ」を基準に指導してください。
    </div>

    <h2 style="color: #1565c0; border-left: 5px solid #1565c0; padding-left: 10px; margin-top: 30px;">１． フォームの５段階分解</h2>

    <div style="margin-bottom: 20px;">
        <h3 style="background: #e3f2fd; padding: 10px; border-radius: 5px; color: #0d47a1;">① スタンス（土台）</h3>
        <p>シュートの安定性はすべて足元で決まります。</p>
        <ul style="padding-left: 20px;">
            <li><strong>足の幅と向き：</strong>肩幅程度。シューティングハンド側の足を半歩前に出す。つま先はリングに対し少し左（10〜20度）へ逃がす（肩の力が抜ける）。</li>
            <li><strong>膝のゆとり：</strong>膝を軽く曲げ、かかとを紙一枚分浮かす意識。母指球に体重を乗せる。</li>
        </ul>
    </div>

    <div style="margin-bottom: 20px;">
        <h3 style="background: #e3f2fd; padding: 10px; border-radius: 5px; color: #0d47a1;">② セットポイント（構え）</h3>
        <p>「どこから打ち始めるか」を固定します。</p>
        <ul style="padding-left: 20px;">
            <li><strong>ショットポケット：</strong>額の右前方（右利きの場合）。</li>
            <li><strong>L字の形成：</strong>肘を90度にし、肩・肘・手首を一直線に。脇を開きすぎない。</li>
            <li><strong>ガイドハンド：</strong>ボールの横に添えるだけ。圧力をかけない。</li>
        </ul>
    </div>

    <div style="margin-bottom: 20px;">
        <h3 style="background: #e3f2fd; padding: 10px; border-radius: 5px; color: #0d47a1;">③ ディップとリフト（連動）</h3>
        <p>ボールを下げてから上げる「リズム」を作ります。</p>
        <ul style="padding-left: 20px;">
            <li><strong>ディップ：</strong>キャッチした位置から、一度おへそ付近までボールを下げる。</li>
            <li><strong>リフト：</strong>「膝が伸びる勢い」をボールに伝える。足が伸び切る直前にセットポイントを通過させる。</li>
        </ul>
    </div>

    <div style="margin-bottom: 20px;">
        <h3 style="background: #e3f2fd; padding: 10px; border-radius: 5px; color: #0d47a1;">④ リリース（放つ）</h3>
        <ul style="padding-left: 20px;">
            <li><strong>肘を高く：</strong>肘が「目の高さ」を超えるまでしっかり上げる（高いアーチを作る）。</li>
            <li><strong>スナップ：</strong>手首を柔らかく返し、バックスピンをかける。</li>
            <li><strong>指先：</strong>最後は人差し指（または中指）から離れる。</li>
        </ul>
    </div>

    <div style="margin-bottom: 20px;">
        <h3 style="background: #e3f2fd; padding: 10px; border-radius: 5px; color: #0d47a1;">⑤ フォロースルー（仕上げ）</h3>
        <ul style="padding-left: 20px;">
            <li><strong>クッキー・ジャー：</strong>高い棚の瓶に手を入れるように、手首を返したまま残す。</li>
            <li><strong>確認：</strong>人差し指がリングの中心を指しているかチェック。</li>
        </ul>
    </div>

    <hr style="margin: 40px 0; border: 0; border-top: 1px solid #eee;">

    <h2 style="color: #2e7d32; border-left: 5px solid #2e7d32; padding-left: 10px; margin-top: 30px;">２． 段階的ドリル（練習メニュー）</h2>
    <p>いきなり遠くから打たせず、以下の順でフォームを固めます。</p>

    <div style="display: flex; flex-wrap: wrap; gap: 15px;">
        <div style="flex: 1 1 300px; border: 1px solid #ddd; padding: 15px; border-radius: 8px;">
            <strong style="color: #2e7d32;">Step 1： 膝つきシュート</strong>
            <p style="font-size: 0.9em; color: #666;">下半身を使わず、上半身の動きだけを確認する。</p>
            <ul style="font-size: 0.9em;">
                <li>片膝立ち、または椅子に座る。</li>
                <li>肘の高さとスナップだけで打つ。</li>
                <li>腕の軌道（真っ直ぐあげる）を覚えるのに最適。</li>
            </ul>
        </div>
        
        <div style="flex: 1 1 300px; border: 1px solid #ddd; padding: 15px; border-radius: 8px;">
            <strong style="color: #2e7d32;">Step 2： ライン・ジャンプ</strong>
            <p style="font-size: 0.9em; color: #666;">真っ直ぐリリーズする感覚を養う。</p>
            <ul style="font-size: 0.9em;">
                <li>コートのライン上に立つ。</li>
                <li>真上にボールを放り、バックスピンをかける。</li>
                <li>落ちてきたボールがライン上に戻れば合格。</li>
            </ul>
        </div>

        <div style="flex: 1 1 300px; border: 1px solid #ddd; padding: 15px; border-radius: 8px;">
            <strong style="color: #2e7d32;">Step 3： フォーム・シューティング</strong>
            <p style="font-size: 0.9em; color: #666;">ゴール下での全身連動確認。</p>
            <ul style="font-size: 0.9em;">
                <li>ゴール下（ノーチャージエリア内）。</li>
                <li>ガイドハンドを使わず、片手だけで打つ。</li>
                <li>足→膝→肘→指先のエネルギー連動を確認。</li>
            </ul>
        </div>
    </div>

    <div style="background-color: #eeeeee; padding: 15px; border-radius: 8px; margin-top: 30px; text-align: center;">
        <strong>⚠️ 指導の注意点</strong><br>
        初心者は「届かせよう」としてフォームを崩します。<br>
        最初は「リングに入らなくていい（届かなくていい）。形とリズムが一定なら100点」と伝えてください。
    </div>
