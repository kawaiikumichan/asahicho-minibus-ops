# コーチング・コード（行動指針）

旭町ミニバスのスタッフは、以下の指針に基づいて指導にあたります。
合言葉は **"Players First"** です。

## 1. 絶対禁止事項（Zero Tolerance）
* **暴力・暴言の禁止**: いかなる理由があっても、身体的暴力、人格を否定する言葉、威圧的な態度は許されません。
* **差別・ハラスメントの禁止**: 選手が萎縮するような指導は行いません。

## 2. 指導のスタイル
### 教えるのではなく「問いかける」
すぐに答え（指示）を出すのではなく、選手自身に考えさせます。
* ❌ 「そこはパスだろ！」
* ⭕ 「今、どういう狙いでドリブルを選んだ？」

### サンドイッチ・フィードバック
改善点を伝えるときは、肯定的な言葉で挟みます。
1. **褒める**: 「ナイスラン！よく走ったね」
2. **改善点**: 「ただ、今のターンはもっと低くできるともっと速くなるよ」
3. **励まし**: 「次はもっと良くなるはず。やってみよう！」

## 3. 保護者との関係
* 我々は保護者のサポーターであり、保護者は一番のファンです。
* 家庭での食事や休息（おにぎり・睡眠）の重要性を伝え、協力体制を築きます。
