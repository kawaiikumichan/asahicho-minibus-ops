# FUN & GUN の定義と真意

旭町ミニバスケットボールクラブの核心となるチーム哲学です。
すべての指導・運営・戦術選択は、この考え方に基づいて決定されます。

## FUN（育成のスタンス）

### ❌ 誤解されがちな意味
* ただヘラヘラと笑って楽しくやること。
* 厳しい練習を避け、楽（ラク）をすること。
* 「勝敗はどうでもいい」と投げやりになること。

### ⭕ 私たちが目指す「真のFUN」
**「上達する喜び（Growth Mindset）」を実感できる場にすること。**

<div style="text-align: center; margin: 20px 0; padding: 15px; background-color: #f8f9fa; border-radius: 8px;">
    なんとか上達しようと挑戦する。<br>
    <span style="font-size: 1.5em; color: #3f51b5;">↓</span><br>
    最初はうまくいかない（失敗・試行錯誤）。<br>
    <span style="font-size: 1.5em; color: #3f51b5;">↓</span><br>
    それでも頑張っているうちに、ふと出来るようになる。<br>
    <span style="font-size: 1.5em; color: #3f51b5;">↓</span><br>
    <strong><span style="font-size: 1.2em; color: #e91e63;">「できた！」という強烈な喜び。</span></strong>
</div>

このプロセス全体を味わうことこそが「FUN」です。

**コーチの役割**
子供たちが壁にぶつかった時、すぐに答えを教えるのではなく、子供自身が壁を乗り越えようとする「挑戦権」を奪わないことです。

---

## GUN（戦術のスタンス）

### ❌ 誤解されがちな意味
* 乱れ打ち（シュートセレクションのない攻撃）。
* 単なるラン＆ガン（ただ走り回るだけ）。

### ⭕ 私たちが目指す「真のGUN」
**「全員攻撃（Total Offense）」の徹底。**

小学生の時期は、経験値も身体能力の伸び方も個人差が非常に大きいです。
目先の勝利だけを優先するなら、早熟な「できる子」にボールを集めて任せるのが最も手っ取り早い方法です。

**しかし、旭町はそれをしません。**

* **全員が主役:** 特定のエースに依存せず、コート上の5人全員がリングを狙います。
* **機会の平等:** 身体能力に関わらず、全員がボールを持ち、判断し、攻める機会を作ります。
* **戦術選択:** そのために、特定のポジションを作らず、全員が広くスペースを使う **[5Out（ファイブ・アウト）](../01_Coaching_Logics/5out.md)** を採用しています。
