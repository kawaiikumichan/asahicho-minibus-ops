# 旭町ミニバスケットボールクラブ チーム運営ガイド 🏀

**「FUN & GUN」で未来をつくる。**
ここは、旭町ミニバスの選手・保護者・コーチ・スタッフ向け運営マニュアル（ナレッジベース）です。
