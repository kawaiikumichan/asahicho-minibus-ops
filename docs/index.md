# 旭町ミニバスケットボールクラブ チーム運営ガイド 🏀

**「FUN & GUN」で未来をつくる。**
ここは、旭町ミニバスの選手・保護者・コーチ・スタッフ向け運営マニュアル（ナレッジベース）です。

        コーチは「答え」を教えるのではなく、選手が「気づく」のを待ちます。
    </div>
    <div style="margin-top: 15px;">
